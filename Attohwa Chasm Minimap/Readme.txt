Which ever map you use, 1024 or 2048 make sure you rename file to be 07_0.gif


Install Location:
..\HorizonXI-Launcher\HorizonXI\Game\

If folder is not created, in the Game folder make a folder - plugins
in the plugins folder create - minimap
in the minimap folder create - maps
The 07_0.gif file goes in the maps folder

If you were already logged in while doing this you may need to /unload minimap then /load minimap
type /mm in game
--> XiPivot Override Settings
  --> type maps for the overlay name
     --> Reload Textures


Enjoy!
