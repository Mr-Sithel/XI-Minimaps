I made these high quality sacrarium maze maps to be used with the minimap plugin to better navigate 
through the maze when day changes. All map images are 1024x1024


Install Location:
..\HorizonXI-Launcher\HorizonXI\Game\

Drop the plugins folder in to the Game folder.
In the Sacrarium Maze Maps folder rename all the .txt extentions to .bat
I did it this way because it seems like windows anti-virus don't like the .bat files.
All the .bat file does is copy and replace the 1c_0.gif with the current day.

(Refrence the minimap.png for example)
Once in game type /mm 
--> XiPivot Override Settings
  --> Type maps in the (Overlay Name)

To use with the minimap double-click the correct day .bat before entering Sacrarium then zone in.
(Example: Earth_Lightning.bat)

If you were already zoned in to sacrarium double-click the correct day .bat then
type /mm in game
--> XiPivot Override Settings
  --> Reload Textures


Enjoy!

Theres also example pictures included.
